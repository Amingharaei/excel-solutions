# Excel Solutions

Practical Excel solutions covering dependent and shrinking data validation, custom number formatting, and conditional formatting automation using Office Scripts.
Each folder is self-contained: it includes the Excel workbook, a README explaining the approach and any constraints worth knowing, and screenshots.

## Index

| Topic | Folder |
|-------|--------|
| Dependent dropdown lists — three approaches using FILTER, XLOOKUP, and named formulas | [dependent-dropdowns](dependent-dropdowns/) |
| Shrinking data validation — tasks disappear from the dropdown as they are assigned | [shrinking-dropdown](shrinking-dropdown/) |
| Custom number formatting and formula-based text formatting side by side | [number-formatting](number-formatting/) |
| Conditional formatting applied via Office Scripts — five scripts from basic K/M/B abbreviation to target-threshold rules | [conditional-formatting-scripts](conditional-formatting-scripts/) |

## Notes

- All solutions that use dynamic array functions (FILTER, UNIQUE, SORT, XLOOKUP, XMATCH) require Excel 365 or Excel 2021 and later.
- The Office Scripts in `conditional-formatting-scripts` require a Microsoft 365 subscription.
